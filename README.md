# 🌿 ECO.happyCleaning.ie

A responsive front-end website based on the design of **ecohappycleaning.ie**, built with modern HTML, SCSS and JavaScript.  
Deployed via GitHub Pages and optimized for clean structure and performance.

🔗 **Live Demo:** https://nastia-pirogova.github.io/ECO.IE/app/



## 📌 About

This repository contains the completed front-end layout and styling for the ECO Happy Cleaning website.  
The project replicates the layout, visual design, and user interface of the original site. It demonstrates skills in responsive HTML structure, SCSS styling, and basic interactivity with JavaScript.



## 🛠️ Tech Stack

- **HTML5** – semantic markup  
- **SCSS** – modular styling  
- **JavaScript** – basic interactions  
- **Gulp** – build automation  
- **GitHub Pages** – hosting



## 🚀 Features

- Fully responsive layout  
- SCSS with modular architecture  
- Gulp build system for CSS/JS workflow  
- Clean code structure and best practices  
- Deployed to GitHub Pages
